# Diamond Cut Motors
A simple responsive website for car showroom which provides elegant browsing experience with image carousels, showroom pages, and basic authentication.

## Feature 
- Simple responsive website
- Simple Authentication form
- Elegant UI inspired by luxury automotive brands
- Images carousels with images and its detail
- Dedicated sections for home, cars showroom, About Us and Login

## Tech Stack
- HTML
- CSS
- Javascript
